# QuestVault Tracker

QuestVault is a browser-based progress tracker for saving and managing question practice files like DSA, SQL, JavaScript, and more.

## Features

- Create separate files for different topics
- Save solved question progress
- Continue from the last solved question
- Add or remove question blocks
- Rename and delete files
- Jump to any question
- Mark all questions as solved
- Clear solved progress
- Progress bar with completion status
- Data saved locally in the browser

## Tech Used

- HTML
- CSS
- JavaScript
- LocalStorage

## Note

Progress is saved in the same browser using LocalStorage. It does not sync across devices.
